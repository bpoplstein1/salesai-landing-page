# Brandon Poplstein

Personal site for **Brandon Poplstein** — AI product leader, co-founder of SalesAi, and AI consultant.

## Pages

- `index.html` — **About / Home**
- `salesai.html` — **SalesAi** landing page
- `consulting.html` — **AI Consulting** (implement AI into product, ops & CX)
- `blog.html` — **Blog** (coming soon)
- `assets/` — images (headshot)

## Tech

Static, self-contained HTML. All CSS and JS are inline; the logo, integration logos, and animated brand mark are embedded directly. Fonts (Roboto + Inter) load from Google Fonts. No build step.

## Live

Served via GitHub Pages. A custom domain (BrandonPoplstein.com) can be pointed at this repo when ready.

## Brand

Mint design system: primary green `#3EDE6F`, dark UI on `#0A0A0C`, Roboto type. "SalesAi" is always written with a lowercase i.
