# SalesAi Landing Page

Marketing landing page for **SalesAi** — the AI workforce for every sales and support conversation.

**Tagline:** Win the moment. Measure the impact.

## What's here

A single, self-contained `index.html`:

- All CSS and JavaScript are inline (no build step, no dependencies to install).
- The SalesAi logo and integration logos are embedded directly (inline SVG and base64), so the file works anywhere on its own.
- The only external resource is the Roboto web font, loaded from Google Fonts.

## View it

- **Locally:** open `index.html` in any modern browser.
- **Live (GitHub Pages):** once Pages is enabled, it will be served at `https://<your-username>.github.io/salesai-landing/`.

## Sections

Hero with a live AI conversation demo, an integrations constellation, a bento "Platform" grid, the three dimensions of impact, an interactive pricing calculator (Volume Plan + Impact Plan = live monthly cost), and a live Impact Health Score dashboard.

## Brand

Built on the SalesAi **Mint** design system: primary green `#3EDE6F`, dark UI on `#0A0A0C`, Roboto type. Always written "SalesAi" (lowercase i).
